## Screens

### 1. 기본 계산기 화면 (Standard Calculator)
- Basic arithmetic operations
- Clean, modern interface
- Dark mode optimized

### 2. 공학 기능 화면 (Scientific Calculator)
- Trigonometric functions (sin, cos, tan)
- Logarithmic functions (ln, log)
- Power and root operations
- Memory functions (MR, M+, M-, MC)
- DEG/RAD mode toggle

### 3. 히스토리 화면 (History)
- Calculation history with date grouping
- Search functionality
- Clear all option
- Bottom navigation

### 4. 단위 변환 화면 (Unit Converter)
- Multiple categories (Length, Mass, Temperature, Time, Volume)
- Real-time conversion
- Swap functionality
- Integrated keypad

## Design System

Extracted from these files:
- **Primary Color**: #4B5EFC
- **Background Dark**: #17171C
- **Surface Dark**: #2E2F38
- **Font**: Space Grotesk (display), Noto Sans (body)
- **Border Radius**: 24px (3xl) for buttons
